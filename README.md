# KIOKUとは、
人間の記憶は24時間で60%忘れると言われています。そんな記憶みたいに、投稿が消えてなくなるアプリです。
他人の目に極力少なくすることで、SNSでの投稿のハードルを下げ、自分だけの投稿一覧に貯めることができます。

## 🌐 App URL
### **https://kioku-1225.herokuapp.com**    

# 開発環境
- macOS 10.14 Mojave
- Rails 6.0.3
- ruby 2.7.2
- bootstrap 5.9
- deivse 

# アプリ機能
* 24時間で消える投稿。
* 投稿一覧は自分だけしかみれません。
* 投稿を見せずに、自分の投稿一覧だけにのせることができます。
* いいね機能だけで、コメントをなくすことでSNSの争いをなくすことができます。
